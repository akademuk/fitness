# ELITE FIT — Преміум шаблон для фітнес-клубу

Цей проект — це енергійний, сучасний та стильний веб-сайт для елітного фітнес-клубу, тренажерного залу або студії персональних тренувань. Він створений, щоб мотивувати, вражати та конвертувати відвідувачів у клієнтів.

## Що отримує клієнт?

- **Драйвовий дизайн:** Поєднання темної естетики, неонових акцентів та якісної типографіки створює атмосферу преміального спортивного клубу.
- **Інтерактивність:**
  - **Героїчний слайдер:** Ефектна презентація основних напрямків (тренування, спа, йога).
  - **Галерея:** Стильна мозаїка з можливістю перегляду фото у повному розмірі (Fancybox).
  - **Тренери:** Картки з інформацією про експертів.
  - **Відгуки:** Слайдер з реальними історіями успіху клієнтів.
- **Функціональність:**
  - **Форма запису:** Зручна форма для бронювання тренувань.
  - **Тарифи:** Чітка презентація клубних карт.
  - **Адаптивність:** Сайт ідеально працює на мобільних пристроях, планшетах та десктопах.
- **Технологічність:** Використання сучасних бібліотек для плавної прокрутки та анімацій робить взаємодію з сайтом приємною та "дорогою".

## Універсальність та Теми

Цей шаблон легко адаптується під будь-який спортивний бренд. Вбудований перемикач тем дозволяє миттєво змінити настрій сайту.

**Вбудовані теми (перемикаються в один клік):**

1.  **Default (Onyx):** Класичний темний стиль з білими та червоними акцентами. Енергія та сила.
2.  **Gold Luxury:** Поєднання чорного та золотого. Для VIP-клубів та ексклюзивних студій.
3.  **Midnight Blue:** Глибокі сині відтінки. Спокій, впевненість, технологічність.
4.  **Pure Light:** Світла, чиста тема. Ідеально для студій йоги, пілатесу або велнес-центрів.
5.  **Forest Calm:** Природні зелені тони. Для еко-фітнесу та тренувань на свіжому повітрі.

Також доступні різні варіанти типографіки (шрифтових пар), від сучасних гротесків до елегантних засічок.

## Технології

- **HTML5 / CSS3:** Семантична верстка, Flexbox/Grid, CSS змінні.
- **JavaScript (ES6+):** Чистий та оптимізований код.
- **GSAP & ScrollTrigger:** Професійні анімації при скролі.
- **Lenis:** Плавна інерційна прокрутка.
- **Swiper:** Потужний слайдер для героїчного блоку та відгуків.
- **Fancybox:** Лайтбокс для перегляду зображень.

Цей сайт — готове рішення для запуску онлайн-присутності вашого фітнес-бізнесу за лічені дні.
